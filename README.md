# Tiến Lên Miền Bắc - Ionic Vue Starter

Offline Tiến Lên Miền Bắc card game starter built with Vue + Ionic.

## Features included

- Landscape layout
- New Game / Continue Game
- 4 players: 1 human + 3 bots
- Easy / Medium / Hard bot setting
- Local ranking: total games, wins, losses
- Local saved game using localStorage
- Basic Tiến Lên Miền Bắc rule engine

## Run

```bash
npm install
npm run dev
```

## Notes

This is an MVP starter. The game engine already supports basic singles, pairs, triples, and same-suit straights. You can extend it with stricter Tiến Lên Miền Bắc rules, stronger bot AI, animations, sound, and Capacitor Android/iOS builds.
